# Introduction
本研究では、東京・神奈川のビジネス街や人口密集エリアを対象に、カフェやファストフード店、コワーキングスペースなどに設置されたスマートフォンやPCの充電用コンセントについて情報共有できるウェブサイトを、OpenStreetMap（OSM）を用いて運用することを目標とした。既存の充電マップでは把握できない「充電可能な席の位置」「駅からの徒歩距離」「混雑時間帯」といった詳細な情報について情報共有できる機能や、アカウント作成によるお気に入り登録機能などのオリジナリティを追加した(Claudeとcodexを併用)。また、自分自身で現地で複数月にわたりフィールドワークを実施し、充電可能な店舗や作業用スペースを記録・分析した。現地での調査結果はピンや説明付きで保管できて(Firebaseを利用)、誰もが自由に共有できる地理空間情報として公開した。本研究は、より細かい実地調査や機能性によって既存データを補完するとともに、データの充実を通じて、外出先で充電場所を必要とする社会人や学生の利便性向上に貢献することを目指す。
