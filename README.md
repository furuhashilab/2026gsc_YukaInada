# Introduction
本研究では、東京・神奈川のビジネス街や人口密集エリアを対象に、カフェやファストフード店、コワーキングスペースなどに設置されたスマートフォン充電用コンセントの利用実態を調査し、OpenStreetMap（OSM）を利用したウェブサイトを運用することを目標とした。既存の充電マップでは把握できない「コンセント席の位置」や「店舗の利用しやすさ」「混雑時間帯」といった詳細な情報について情報共有できるオリジナルサイトを作成した(Claudeとcodexを併用)。現地で複数日にわたりフィールドワークを実施し、時系列で記録・分析した。現地での調査結果はピンや説明付きで保管できるようにし(Firebaseを利用)、誰もが自由に共有できる地理空間情報として公開した。本研究は、利用者目線の実地調査によって既存データを補完するとともに、オープンデータの充実を通じて、外出先で充電場所を必要とする社会人や学生の利便性向上に貢献することを目指す。
